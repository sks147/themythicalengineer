## CalcTrack

CalcTrack is a notepad-style calculator for iOS and Android. Type natural math expressions line by line in a text editor and see results appear instantly in a right-aligned answer column. Supports multi-currency conversions with live exchange rates, unit conversions, date arithmetic, variables, functions, and more. Available in a free version (with ads) and an ad-free Pro version.

### Features

#### Calculator

- Multi-line text editor where each line is evaluated as a math expression
- Right-aligned answer column with live results
- Basic arithmetic, percentages ("15% of 490", "500 + 10%"), and parentheses
- Variables (name = expression) and line references
- Built-in functions: sqrt, round, floor, ceil, abs, min, max
- SI scalars: k, M, B, T
- Subtotals and totals with heading boundaries
- Syntax highlighting

#### Currency & Units

- Multi-currency support (USD, EUR, GBP, INR, JPY) with live exchange rates
- Unit conversions across length, mass, volume, temperature, time, data storage, and speed

#### Date Arithmetic

- Date keywords: "today", "tomorrow"
- Date calculations: "days until Christmas", "date + 2 weeks"

#### Countdown Events

- Countdown events for future, past, and recurring dates with live-ticking cards
- Android home-screen widget for countdown events

#### Organization

- Multiple sheets with customizable icons and colors
- Workspaces for grouping sheets and events
